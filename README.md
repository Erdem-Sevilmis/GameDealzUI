# DealZ

A Flutter application for finding the best deals on games.

## About

This project was created to provide a simple and efficient way for gamers to find the best prices for their favorite games across multiple online stores. The motivation behind this project was to create a one-stop solution for game deals, saving users time and money.

## Features

- Search for games
- View game details and prices from various key sites
- Add games to your favorites list
- Light and dark theme support

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/dealz.git
   ```
2. **Install dependencies:**
   ```bash
   flutter pub get
   ```
3. **Run the app:**
   ```bash
   flutter run
   ```
